O projeto desenvolvido consiste em um jogo 2D denominado "Fruit Hunter", criado utilizando a linguagem Python e a biblioteca Pygame. O jogador controla um personagem através das teclas W, A, S e D, tendo como objetivo coletar 15 frutas distribuídas aleatoriamente pelo cenário antes que o tempo limite de 45 segundos seja atingido. O jogo possui menu inicial, sistema de pontuação, cronômetro, condição de vitória e condição de derrota, atendendo aos requisitos propostos pela disciplina de Linguagem de Programação Aplicada.
